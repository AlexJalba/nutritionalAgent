# Skills Index

Load the skill that matches your task. Skills are focused and fast to load.
Never load all skills at once — load only what you need.

## When to load which skill

| You need to... | Load |
|---------------|------|
| Start a feature / fix a bug | `../03-workflow.md` first, then this index |
| Create/update/search Jira issues | `jira/SKILL.md` — **mandatory before any Jira operation** |
| Read/write SAP Wiki pages | `wiki/SKILL.md` |
| Access GitHub repos/PRs/files | `github/SKILL.md` |
| Run Phase 4 audit | `review/SKILL.md` |
| Close a sprint / end of feature | `sprint-close/SKILL.md` |

## What each skill contains

**jira/SKILL.md** — SAP Jira auth decision tree, MCP failure modes, raw REST fallback recipes, field ID reference
**wiki/SKILL.md** — SAP Wiki auth, MCP vs REST decision, page read/write patterns
**github/SKILL.md** — github.tools.sap vs github.wdf.sap.corp, cookie auth, gh CLI fallback
**review/SKILL.md** — Phase 4 full audit: all 9 personas + SAP Product Standards validation
**sprint-close/SKILL.md** — archiving, CURRENT.md maintenance, brain update checklist
